# GenAI-Projects26

MkDocs-basierte Dokumentationsseite für die praktische Projektarbeit im Kurs **„KI in Medienanwendungen“** (Sommersemester 2026).

## Quickstart

```bash
pip install mkdocs mkdocs-material
mkdocs serve        # Lokaler Server
mkdocs build        # Statische Seite bauen
mkdocs gh-deploy    # Auf GitHub Pages deployen
```
