# Projekt 3 — Interaktive KI-Anwendung

## Thema

Entwicklung einer interaktiven Anwendung, die KI-Technologien in Echtzeit nutzt.

## Ziele

- Konzeption einer interaktiven Medienanwendung
- Echtzeit-KI-Verarbeitung (z. B. Webcam, Mikrofon)
- Integration von KI-Modellen in eine lauffähige Anwendung
- Nutzerfreundliches Interface

## Mögliche Ansätze

- Echtzeit-Stiltransfer über Webcam
- Interaktive Bildgenerierung mit Nutzereingabe
- Sprachgesteuerte Medienanwendung
- KI-gestützte immersive Installation

## Tools und Ressourcen

| Tool | Verwendung |
|------|-----------|
| Python / Flask / FastAPI | Backend |
| JavaScript / p5.js | Frontend |
| TensorFlow.js / ONNX Runtime | Echtzeit-Inferenz |
| Streamlit / Gradio | Schnelle UI-Prototypen |
