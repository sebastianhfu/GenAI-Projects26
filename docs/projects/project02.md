# Projekt 2 — Audio- und Videogenerierung

## Thema

Erzeugung von Audio- und Videoinhalten mit generativer KI.

## Ziele

- Einarbeitung in Audio-Synthese-Tools (z. B. Suno, Udio, AudioCraft)
- Erzeugung von KI-gestützten Video-Sequenzen
- Synchronisation von Audio und Video
- Entwicklung eines durchgängigen Medienprodukts

## Tools und Ressourcen

| Tool | Verwendung |
|------|-----------|
| Suno / Udio | Musikgenerierung |
| Runway / Pika | Videogenerierung |
| FFmpeg | Post-Production |
| DaVinci Resolve | Schnitt und Nachbearbeitung |
